# Copy Images Instructions

To complete the image setup, please copy these two images to the `public/assets/images/` directory:

```bash
cp TorqueAscmo.png public/assets/images/
cp TorqueModel_opt.jpg public/assets/images/
```

Or manually copy:
- `TorqueAscmo.png` → `public/assets/images/TorqueAscmo.png`
- `TorqueModel_opt.jpg` → `public/assets/images/TorqueModel_opt.jpg`

These images will be displayed as thumbnails in the calibration suite orchestrator, and clicking on them will open a fullscreen modal view.
